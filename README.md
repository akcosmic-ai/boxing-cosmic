# The 30-Minute Bag Program

A self-contained, single-file boxing training site. Live at **https://akcosmic-ai.github.io/boxing-cosmic/**

- 30-minute home heavy-bag program with two levels: Foundation and Advanced sessions
- Built-in round timer: six 3:00 rounds with 1:00 rest, audio bells, per-round focus prompts
- Voice coach mode: calls random combos and defensive commands out loud during work rounds
- Daily punch task that rotates by weekday, with one-tap completion
- Progress tracking in localStorage: day streak, weekly session goal, phase progression, 8-week heatmap
- Low-impact, joint-friendly footwork rules (flat-footed step-drag, no bouncing or jumping)
- Embedded, verified technique videos for every move (jab, cross, hooks, uppercuts, slips, rolls, pivots, check hook)
- SVG form diagrams: stance, step-drag, pivot, angles, slip, roll
- Self-defense module: principles that transfer from boxing to real situations, with videos
- Screen wake-lock during sessions so the phone stays on at the bag

No build step, no dependencies, no accounts. Everything lives in `index.html`. Progress data never leaves the browser.
